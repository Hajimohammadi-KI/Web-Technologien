مقدمه توزیع‌های پیوسته توزیع یکنواخت توزیع پارتو توزیع نمایی توزیع نرمال توزیع هندسی توزیع دوجمله‌ای قضیه حد مرکزی تمرین‌های توتوریوم تمرین‌های کاربردی نمای کلی فایل‌ها

دوره کامل آمار و احتمال

یادگیری توزیع‌های احتمال، کاربردها و حل مسائل

مقدمه

واژگان کلیدی: Statistics = آمار، Probability = احتمال، Distribution = توزیع، Variance = واریانس، Mean = میانگین

اهداف دوره:

در این دوره، مفاهیم کلیدی آمار و احتمال، شامل انواع توزیع‌های احتمال، کاربردهای عملی آن‌ها در دنیای واقعی و روش‌های حل مسائل عملی را یاد خواهید گرفت.

  • توزیع‌های پیوسته: نرمال، یکنواخت، نمایی، پارتو
  • توزیع‌های گسسته: دوجمله‌ای، هندسی، پواسون
  • کاربردهای عملی: مسائل واقعی از حوزه‌های مختلف
  • شبیه‌سازی: استفاده از R برای حل مسائل
  • تمرین‌های توتوریوم: تقویت مفاهیم با تمرین‌های عملی

ساختار دوره

این دوره شامل موارد زیر است:

  • مبانی نظری هر توزیع با فرمول‌های کامل
  • مثال‌های عملی و کاربردی از مسائل واقعی
  • کدهای R برای شبیه‌سازی و محاسبات
  • تمرین‌های حل شده گام به گام
  • تمرین‌های توتوریوم با حل کامل
  • مسائل چالش‌برانگیز برای تقویت مهارت

فرمول‌های کلیدی آمار:

میانگین: μ = E(X)

واریانس: σ² = Var(X) = E[(X-μ)²]

انحراف معیار: σ = √Var(X)

ضریب تغییرات: CV = σ/μ

توزیع‌های پیوسته

تعریف توزیع پیوسته

توزیع احتمال پیوسته توزیعی است که متغیر تصادفی آن می‌تواند هر مقدار حقیقی در یک بازه را اتخاذ کند. این توزیع‌ها با تابع چگالی احتمال (PDF) مشخص می‌شوند.

خصوصیات کلیدی توزیع‌های پیوسته:

تابع چگالی احتمال (PDF): f(x) ≥ 0 برای همه x

مساحت کل زیر منحنی: ∫_{-∞}^{∞} f(x)dx = 1

تابع توزیع تجمعی (CDF): F(x) = P(X ≤ x) = ∫_{-∞}^{x} f(t)dt

احتمال نقطه‌ای: P(X = x₀) = 0

احتمال بازه‌ای: P(a ≤ X ≤ b) = ∫_a^b f(x)dx

انواع توزیع‌های پیوسته که در این دوره بررسی می‌شوند:

  • توزیع یکنواخت (Uniform): احتمال یکسان در بازه مشخص
  • توزیع نرمال (Normal): منحنی زنگوله‌ای، پرکاربردترین توزیع
  • توزیع نمایی (Exponential): مدل‌سازی زمان‌های انتظار
  • توزیع پارتو (Pareto): مدل‌سازی درآمد و ثروت

توزیع یکنواخت (Uniform Distribution)

تعریف توزیع یکنواخت

توزیع یکنواخت در بازه [a, b] به این معنی است که همه نقاط در این بازه احتمال یکسانی دارند. این ساده‌ترین نوع توزیع پیوسته است.

نماد: X ~ U(a, b)

فرمول‌های کلیدی توزیع یکنواخت:

تابع چگالی: f(x) = 1/(b-a) برای a ≤ x ≤ b، و 0 در غیر این صورت

تابع توزیع: F(x) = (x-a)/(b-a) برای a ≤ x ≤ b

میانگین: μ = E(X) = (a+b)/2

واریانس: σ² = Var(X) = (b-a)²/12

انحراف معیار: σ = (b-a)/(2√3)

میانه: median = (a+b)/2

مثال کاربردی: زمان انتظار در سوپرمارکت

زمان کل انتظار در سوپرمارکت بین 8.5 تا 11 دقیقه توزیع یکنواخت دارد.

سوال‌ها:

الف) احتمال انتظار حداکثر 9 دقیقه چقدر است؟

ب) بازه مرکزی که 68% احتمال را شامل می‌شود چیست؟

ج) میانگین و انحراف معیار زمان انتظار؟

چرا توزیع یکنواخت؟ زمان ورود تصادفی به سوپرمارکت باعث می‌شود هر لحظه از بازه انتظار احتمال یکسانی داشته باشد.

حل کامل مسئله:

X ~ U(8.5, 11)

الف) P(X ≤ 9) = (9-8.5)/(11-8.5) = 0.5/2.5 = 0.2 = 20%

ب) μ = (8.5+11)/2 = 9.75, σ = (11-8.5)/(2√3) ≈ 0.72

بازه مرکزی: [μ-σ, μ+σ] = [9.03, 10.47]

ج) میانگین = 9.75 دقیقه، انحراف معیار = 0.72 دقیقه

# توزیع یکنواخت در R - Uniform Distribution in R

a <- 8.5  # حد پایین - Lower bound
b <- 11   # حد بالا - Upper bound

# محاسبه احتمال - Probability calculation
prob_less_9 <- punif(9, min = a, max = b)
print(paste("P(X ≤ 9) =", round(prob_less_9, 3)))

# محاسبه آمارگان - Statistics calculation
mean_uniform <- (a + b) / 2
var_uniform <- (b - a)^2 / 12
sd_uniform <- sqrt(var_uniform)

print(paste("میانگین:", round(mean_uniform, 2), "دقیقه"))
print(paste("واریانس:", round(var_uniform, 2)))
print(paste("انحراف معیار:", round(sd_uniform, 2), "دقیقه"))

# محاسبه کوانتیل‌ها - Quantiles calculation
q25 <- qunif(0.25, min = a, max = b)
q50 <- qunif(0.50, min = a, max = b)
q75 <- qunif(0.75, min = a, max = b)

print(paste("Q1 (25%):", round(q25, 2)))
print(paste("Q2 (50%):", round(q50, 2)))
print(paste("Q3 (75%):", round(q75, 2)))

# رسم نمودار - Plotting
x <- seq(a-1, b+1, 0.1)
y <- dunif(x, min = a, max = b)

plot(x, y, type = "l", lwd = 3, col = "blue",
     main = "توزیع یکنواخت - زمان انتظار در سوپرمارکت", 
     xlab = "زمان (دقیقه)", ylab = "چگالی احتمال f(x)",
     xlim = c(7, 12), ylim = c(0, 0.5))

# نمایش میانگین و انحراف معیار
abline(v = mean_uniform, col = "red", lty = 2, lwd = 2)
abline(v = c(mean_uniform - sd_uniform, mean_uniform + sd_uniform), 
       col = "green", lty = 3, lwd = 2)

legend("topright",

       col = c("blue", "red", "green"), 
       lty = c(1, 2, 3), lwd = 2)

توزیع پارتو (Pareto Distribution)

کاربردهای توزیع پارتو

توزیع پارتو برای مدل‌سازی توزیع درآمد، اندازه شهرها، و پدیده‌هایی که قانون 80-20 در آن‌ها صدق می‌کند، استفاده می‌شود. این توزیع ویژگی "دم سنگین" دارد.

نماد: X ~ Pareto(x_m, α)

فرمول‌های کلیدی توزیع پارتو:

تابع چگالی: f(x) = (α × x_m^α) / x^(α+1) برای x ≥ x_m

تابع توزیع: F(x) = 1 - (x_m/x)^α

میانگین: μ = (α × x_m)/(α-1) برای α > 1

واریانس: σ² = (α × x_m²)/[(α-1)²(α-2)] برای α > 2

میانه: median = x_m × 2^(1/α)

مثال عملی: توزیع درآمد در یک شهر

فرض کنید درآمد در یک شهر از توزیع پارتو با x_m = 30000 تومان و α = 1.5 پیروی می‌کند.

سوال‌ها:

الف) چه درصد از مردم درآمد بیشتر از 50000 تومان دارند؟

ب) میانگین درآمد چقدر است؟

ج) درآمدی که 20% افراد بالاتر از آن دارند چقدر است؟

حل کامل:

الف) P(X > 50000) = (x_m/x)^α = (30000/50000)^1.5 = (0.6)^1.5 ≈ 0.464

پاسخ: حدود 46.4% از مردم درآمد بیشتر از 50000 تومان دارند.

ب) μ = (α × x_m)/(α-1) = (1.5 × 30000)/(1.5-1) = 45000/0.5 = 90000 تومان

ج) P(X > x) = 0.2 ⟹ (30000/x)^1.5 = 0.2

x = 30000 / (0.2)^(1/1.5) ≈ 82156 تومان

# توزیع پارتو در R - Pareto Distribution in R

# نصب و بارگذاری کتابخانه - Install and load library
# install.packages("VGAM")
library(VGAM)

# پارامترهای توزیع - Distribution parameters
x_m <- 30000  # حد پایین - Minimum value
alpha <- 1.5  # پارامتر شکل - Shape parameter

# الف) احتمال درآمد بیش از 50000 - Probability of income > 50000
prob_greater_50k <- 1 - ppareto(50000, scale = x_m, shape = alpha)
print(paste("P(X > 50000) =", round(prob_greater_50k, 3)))
print(paste("درصد افراد با درآمد بالای 50000:", round(prob_greater_50k * 100, 1), "%"))

# ب) میانگین درآمد - Mean income
if(alpha > 1) {
    mean_income <- alpha * x_m / (alpha - 1)
    print(paste("میانگین درآمد:", round(mean_income, 0), "تومان"))
} else {
    print("میانگین وجود ندارد (α ≤ 1)")
}

# ج) کوانتیل 80% (20% بالاتر از این مقدار) - 80th percentile
income_80th <- qpareto(0.8, scale = x_m, shape = alpha)
print(paste("80% مردم درآمد کمتر از", round(income_80th, 0), "تومان دارند"))

# محاسبه میانه - Median calculation
median_income <- x_m * (2^(1/alpha))
print(paste("میانه درآمد:", round(median_income, 0), "تومان"))

# رسم نمودار - Plotting
x_range <- seq(x_m, 150000, 1000)
y_density <- dpareto(x_range, scale = x_m, shape = alpha)

plot(x_range, y_density, type = "l", lwd = 2, col = "purple",
     main = "توزیع پارتو - توزیع درآمد", 
     xlab = "درآمد (تومان)", ylab = "چگالی احتمال",
     xlim = c(30000, 120000))

# نمایش آمارگان مهم
abline(v = mean_income, col = "red", lty = 2, lwd = 2)
abline(v = median_income, col = "blue", lty = 3, lwd = 2)
abline(v = income_80th, col = "green", lty = 4, lwd = 2)

legend("topright", 
       c("چگالی پارتو", "میانگین", "میانه", "کوانتیل 80%"),
       col = c("purple", "red", "blue", "green"), 
       lty = c(1, 2, 3, 4), lwd = 2)

# نمایش قانون 80-20 - Show 80-20 rule
wealth_80_percent <- qpareto(0.8, scale = x_m, shape = alpha)
print(paste("قانون 80-20: 20% ثروتمندان درآمد بالای", 
           round(wealth_80_percent, 0), "دارند"))

توزیع نمایی (Exponential Distribution)

کاربردهای توزیع نمایی

توزیع نمایی برای مدل‌سازی زمان‌های انتظار، طول عمر تجهیزات، فاصله زمانی بین رویدادها، و سیستم‌هایی با ویژگی "بدون حافظه" استفاده می‌شود.

نماد: X ~ Exp(λ)

فرمول‌های کلیدی توزیع نمایی:

تابع چگالی: f(x) = λe^(-λx) برای x ≥ 0

تابع توزیع: F(x) = 1 - e^(-λx)

میانگین: μ = E(X) = 1/λ

واریانس: σ² = Var(X) = 1/λ²

انحراف معیار: σ = 1/λ

میانه: median = ln(2)/λ ≈ 0.693/λ

ویژگی بدون حافظه: P(X > s+t | X > s) = P(X > t)

مسئله: زمان گارانتی کامپیوتر

شرکتی چهار کامپیوتر تولید کرده که زمان‌های خرابی آن‌ها عبارتند از:

داده‌ها: 1200، 1900، 3200، 2400 ساعت

سوال‌ها:

الف) برآورد پارامتر λ از روی داده‌ها

ب) احتمال خرابی در 1000 ساعت اول

ج) زمان گارانتی برای 10% خرابی

د) احتمال اینکه کامپیوتر بیش از 2000 ساعت کار کند

حل کامل مسئله:

الف) برآورد پارامتر:

میانگین = (1200+1900+3200+2400)/4 = 2175 ساعت

λ̂ = 1/میانگین = 1/2175 ≈ 0.000460

ب) P(X ≤ 1000):

P(X ≤ 1000) = 1 - e^(-λ×1000) = 1 - e^(-0.000460×1000) ≈ 0.368

ج) زمان برای 10% خرابی:

0.1 = 1 - e^(-λt) ⟹ t = -ln(0.9)/λ ≈ 229 ساعت

د) P(X > 2000):

P(X > 2000) = e^(-λ×2000) = e^(-0.000460×2000) ≈ 0.398

# حل کامل مسئله توزیع نمایی - Complete exponential distribution solution

failure_times <- c(1200, 1900, 3200, 2400)  # زمان‌های خرابی

# 2. برآورد پارامتر - Parameter estimation
mean_time <- mean(failure_times)
lambda_hat <- 1/mean_time

print("=== برآورد پارامترها ===")
print(paste("میانگین زمان خرابی:", mean_time, "ساعت"))
print(paste("برآورد λ:", round(lambda_hat, 6)))
print(paste("انحراف معیار:", round(1/lambda_hat, 1), "ساعت"))

# 3. ب) احتمال خرابی در 1000 ساعت
prob_failure_1000 <- pexp(1000, rate = lambda_hat)
print(paste("\nP(خرابی در 1000 ساعت) =", round(prob_failure_1000, 3)))

# 4. ج) زمان گارانتی برای 10% خرابی
warranty_time <- qexp(0.1, rate = lambda_hat)
print(paste("زمان گارانتی 10%:", round(warranty_time, 0), "ساعت"))

# 5. د) احتمال کار بیش از 2000 ساعت
prob_survival_2000 <- 1 - pexp(2000, rate = lambda_hat)
print(paste("P(کار بیش از 2000 ساعت) =", round(prob_survival_2000, 3)))

# 6. محاسبه میانه - Median calculation
median_time <- log(2) / lambda_hat
print(paste("میانه زمان خرابی:", round(median_time, 0), "ساعت"))

# 7. رسم نمودار توزیع
x_vals <- seq(0, 8000, 100)

# نمایش آمارگان مهم
abline(v = mean_time, col = "red", lty = 2, lwd = 2)

abline(v = warranty_time, col = "orange", lty = 4, lwd = 2)

legend("topright", 
       c("تابع بقا", "میانگین", "میانه", "گارانتی 10%"),

توزیع نرمال (Normal Distribution)

اهمیت توزیع نرمال

توزیع نرمال پرکاربردترین توزیع در آمار است و بنیان بسیاری از روش‌های آماری را تشکیل می‌دهد. شکل آن مانند زنگوله است، متقارن حول میانگین و دارای خصوصیات منحصربه‌فردی است.

نماد: X ~ N(μ, σ²)

فرمول‌های کلیدی توزیع نرمال:

تابع چگالی: f(x) = (1/σ√(2π)) × e^(-(x-μ)²/2σ²)

میانگین: E(X) = μ

واریانس: Var(X) = σ²

انحراف معیار: SD(X) = σ

استاندارد: Z = (X-μ)/σ ~ N(0,1)

قانون 68-95-99.7: 68% در [μ±σ]، 95% در [μ±2σ]، 99.7% در [μ±3σ]

مسئله جامع: وزن پرتقال‌ها

وزن پرتقال‌ها از توزیع نرمال با میانگین 230 گرم و انحراف معیار 25 گرم پیروی می‌کند.

سوال‌ها:

الف) احتمال وزن بیش از 250 گرم؟

ب) 95% پرتقال‌ها کمتر از چه وزنی دارند؟

ج) اگر 9 پرتقال را بسته‌بندی کنیم، احتمال وزن کل بیش از 2 کیلو؟

د) چه درصد پرتقال‌ها در بازه [205, 255] گرم قرار دارند؟

چرا توزیع نرمال؟ وزن پرتقال‌ها تحت تأثیر عوامل متعددی مثل ژنتیک، آب و هوا، و خاک است که بر اساس قضیه حد مرکزی به توزیع نرمال منجر می‌شود.

حل کامل مسئله:

X ~ N(230, 25²)

الف) P(X > 250):

Z = (250-230)/25 = 0.8

P(X > 250) = P(Z > 0.8) = 1 - Φ(0.8) ≈ 0.212

ب) صدک 95:

P(X ≤ x) = 0.95 ⟹ x = μ + σ×z₀.₉₅ = 230 + 25×1.645 ≈ 271 گرم

ج) وزن کل 9 پرتقال:

S = X₁ + X₂ + ... + X₉ ~ N(9×230, √9×25) = N(2070, 75)

P(S > 2000) = P(Z > (2000-2070)/75) = P(Z > -0.93) ≈ 0.824

د) P(205 ≤ X ≤ 255):

این بازه μ±σ است، پس طبق قانون 68-95-99.7 حدود 68% پرتقال‌ها در این بازه قرار دارند.

توزیع هندسی (Geometric Distribution)

تعریف توزیع هندسی

توزیع هندسی تعداد آزمایش‌های مستقل با احتمال موفقیت ثابت تا اولین موفقیت را مدل می‌کند. این توزیع خاصیت "بدون حافظه" دارد.

نماد: X ~ Geo(p)

فرمول‌های کلیدی توزیع هندسی:

تابع احتمال: P(X = k) = (1-p)^(k-1) × p برای k = 1, 2, 3, ...

تابع توزیع: P(X ≤ k) = 1 - (1-p)^k

میانگین: E(X) = 1/p

واریانس: Var(X) = (1-p)/p²

انحراف معیار: σ = √(1-p)/p

ویژگی بدون حافظه: P(X > m+n | X > m) = P(X > n)

مسئله: پرتاب سکه تا اولین شیر

سکه‌ای منصفانه پرتاب می‌شود تا اولین بار شیر بیاید.

سوال‌ها:

الف) میانگین و انحراف معیار تعداد پرتاب‌ها؟

ب) احتمال حداکثر 3 پرتاب؟

ج) اگر در 2 پرتاب اول شیر نیامده، احتمال در پرتاب سوم شیر بیاید؟

د) شبیه‌سازی 100 نمونه در R

چرا توزیع هندسی؟ ما آزمایش‌های مستقل (پرتاب سکه) تا اولین موفقیت (شیر) شمارش می‌کنیم که دقیقاً تعریف توزیع هندسی است.

حل کامل مسئله:

X ~ Geo(p = 0.5)

الف) E(X) = 1/0.5 = 2، Var(X) = 0.5/0.25 = 2، σ = √2 ≈ 1.41

انتظار داریم در متوسط 2 پرتاب برای اولین شیر لازم باشد.

ب) P(X ≤ 3) = 1 - (0.5)³ = 1 - 0.125 = 0.875

احتمال 87.5% که در حداکثر 3 پرتاب شیر بیاید.

ج) به دلیل خاصیت بدون حافظه: P(X = 3 | X > 2) = P(X = 1) = 0.5

د) کد شبیه‌سازی در بخش زیر ارائه شده است.

توزیع دوجمله‌ای (Binomial Distribution)

تعریف توزیع دوجمله‌ای

توزیع دوجمله‌ای تعداد موفقیت‌ها در n آزمایش مستقل برنولی را مدل می‌کند که هر کدام احتمال موفقیت p دارند.

نماد: X ~ Bin(n, p)

شرایط: آزمایش‌های مستقل، احتمال ثابت، تعداد ثابت آزمایش

فرمول‌های کلیدی توزیع دوجمله‌ای:

تابع احتمال: P(X = k) = C(n,k) × p^k × (1-p)^(n-k)

ضریب دوجمله‌ای: C(n,k) = n! / (k!(n-k)!)

میانگین: E(X) = np

واریانس: Var(X) = np(1-p)

انحراف معیار: σ = √(np(1-p))

تقریب نرمال: اگر np ≥ 5 و n(1-p) ≥ 5

مسئله: آزمون چهارگزینه‌ای

دانش‌آموزی آزمون 15 سؤالی چهارگزینه‌ای دارد و به صورت تصادفی پاسخ می‌دهد.

سوال‌ها:

الف) احتمال حداکثر 4 پاسخ صحیح؟

ب) میانگین و انحراف معیار تعداد پاسخ‌های صحیح؟

ج) آیا تقریب نرمال قابل استفاده است؟

د) احتمال بین 2 تا 6 پاسخ صحیح؟

چرا توزیع دوجمله‌ای؟ تعداد ثابت آزمایش مستقل (15 سؤال)، هر آزمایش دو نتیجه (درست/غلط)، احتمال ثابت موفقیت (p=0.25).

حل کامل مسئله:

X ~ Bin(15, 0.25)

الف) P(X ≤ 4) = ∑_{k=0}^{4} C(15,k) × (0.25)^k × (0.75)^(15-k) ≈ 0.686

ب) E(X) = 15×0.25 = 3.75، σ = √(15×0.25×0.75) ≈ 1.68

ج) np = 3.75 < 5 و n(1-p) = 11.25 ≥ 5

تقریب نرمال ایده‌آل نیست زیرا np < 5

د) P(2 ≤ X ≤ 6) = P(X ≤ 6) - P(X ≤ 1) ≈ 0.943 - 0.080 = 0.863

# حل کامل مسئله توزیع دوجمله‌ای - Complete binomial distribution solution

# پارامترهای توزیع
n <- 15      # تعداد سؤالات
p <- 0.25    # احتمال پاسخ صحیح (1/4)

print("=== پارامترهای توزیع ===")
print(paste("تعداد آزمایش n =", n))
print(paste("احتمال موفقیت p =", p))
print(paste("احتمال شکست q = 1-p =", 1-p))

# ب) میانگین و انحراف معیار
mean_binom <- n * p
var_binom <- n * p * (1 - p)
sd_binom <- sqrt(var_binom)

print(paste("\n=== ب) آمارگان ==="))
print(paste("میانگین E(X) =", mean_binom))
print(paste("واریانس Var(X) =", var_binom))
print(paste("انحراف معیار σ =", round(sd_binom, 2)))

# الف) احتمال حداکثر 4 پاسخ صحیح
prob_at_most_4 <- pbinom(4, size = n, prob = p)
print(paste("\n=== الف) P(X ≤ 4) ==="))
print(paste("P(X ≤ 4) =", round(prob_at_most_4, 3)))

# محاسبه احتمالات جزئی
for(k in 0:6) {
    prob_k <- dbinom(k, size = n, prob = p)
    print(paste("P(X =", k, ") =", round(prob_k, 4)))
}

# ج) بررسی شرایط تقریب نرمال
condition1 <- n * p
condition2 <- n * (1 - p)
print(paste("\n=== ج) بررسی تقریب نرمال ==="))
print(paste("np =", condition1, "(باید ≥ 5)"))
print(paste("n(1-p) =", condition2, "(باید ≥ 5)"))

if(condition1 >= 5 && condition2 >= 5) {
    print("تقریب نرمال قابل استفاده است")
    # تقریب نرمال با تصحیح پیوستگی
    prob_4_normal <- pnorm(4.5, mean = mean_binom, sd = sd_binom)
    print(paste("P(X ≤ 4) با تقریب نرمال:", round(prob_4_normal, 3)))
} else {
    print("تقریب نرمال ایده‌آل نیست (np < 5)")
}

# د) احتمال بین 2 تا 6 پاسخ صحیح
prob_between_2_6 <- pbinom(6, n, p) - pbinom(1, n, p)
print(paste("\n=== د) P(2 ≤ X ≤ 6) ==="))
print(paste("P(2 ≤ X ≤ 6) =", round(prob_between_2_6, 3)))
print(paste("P(X ≤ 6) =", round(pbinom(6, n, p), 3)))
print(paste("P(X ≤ 1) =", round(pbinom(1, n, p), 3)))

# محاسبات اضافی
prob_exactly_mean <- dbinom(round(mean_binom), n, p)
prob_above_mean <- 1 - pbinom(floor(mean_binom), n, p)
print(paste("\nاحتمال دقیقاً برابر میانگین:", round(prob_exactly_mean, 3)))
print(paste("احتمال بالاتر از میانگین:", round(prob_above_mean, 3)))

# رسم نمودارهای مختلف
par(mfrow = c(2, 2))

# 1. تابع احتمال
k_values <- 0:n
prob_values <- dbinom(k_values, size = n, prob = p)
barplot(prob_values, names.arg = k_values,
        main = "تابع احتمال دوجمله‌ای",
        xlab = "تعداد پاسخ‌های صحیح (k)", ylab = "P(X = k)",
        col = "lightcoral", border = "darkred")

# نمایش میانگین
abline(v = mean_binom + 0.5, col = "blue", lwd = 2, lty = 2)

# 2. تابع توزیع تجمعی
cdf_values <- pbinom(k_values, size = n, prob = p)
plot(k_values, cdf_values, type = "s", lwd = 2, col = "darkgreen",
     main = "تابع توزیع تجمعی",
     xlab = "k", ylab = "P(X ≤ k)")
grid()
abline(h = c(0.25, 0.5, 0.75), col = "gray", lty = 3)

# 3. مقایسه با تقریب نرمال (اگر امکان‌پذیر باشد)
if(condition1 >= 5 && condition2 >= 5) {
    x_continuous <- seq(0, n, 0.1)
    y_normal <- dnorm(x_continuous, mean = mean_binom, sd = sd_binom)
    
    plot(k_values, prob_values, type = "h", lwd = 3, col = "red",
         main = "مقایسه دوجمله‌ای و نرمال",
         xlab = "k", ylab = "احتمال/چگالی")
    lines(x_continuous, y_normal, col = "blue", lwd = 2)
    legend("topright", c("دوجمله‌ای", "تقریب نرمال"), 
           col = c("red", "blue"), lty = c(1, 1), lwd = c(3, 2))
} else {
    # نمایش محدوده اطمینان
    lower_bound <- max(0, mean_binom - 2*sd_binom)
    upper_bound <- min(n, mean_binom + 2*sd_binom)
    
    barplot(prob_values, names.arg = k_values,
            main = "محدوده 95% (μ ± 2σ)",
            xlab = "k", ylab = "P(X = k)",
            col = ifelse(k_values >= lower_bound & k_values <= upper_bound, 
                        "lightblue", "lightgray"))
    abline(v = c(lower_bound, upper_bound) + 0.5, col = "red", lwd = 2)
}

# 4. شبیه‌سازی
set.seed(123)
simulated_results <- rbinom(1000, size = n, prob = p)
hist(simulated_results, breaks = -0.5:(n+0.5), freq = FALSE,
     main = "شبیه‌سازی (1000 آزمون)",
     xlab = "تعداد پاسخ‌های صحیح", ylab = "فراوانی نسبی",
     col = "lightyellow", border = "orange")

# اضافه کردن تابع احتمال تئوری
points(k_values, prob_values, col = "red", pch = 16, cex = 1.2)
lines(k_values, prob_values, col = "red", lwd = 2)

# بازیابی تنظیمات نمودار

# نتایج شبیه‌سازی
print(paste("\n=== نتایج شبیه‌سازی ==="))
print(paste("میانگین شبیه‌سازی:", round(mean(simulated_results), 2)))
print(paste("انحراف معیار شبیه‌سازی:", round(sd(simulated_results), 2)))
print(paste("درصد نمونه‌هایی که ≤ 4 پاسخ صحیح:", 
           round(mean(simulated_results <= 4) * 100, 1), "%"))

# مقایسه کوانتیل‌ها

quantiles_sim <- quantile(simulated_results, c(0.25, 0.5, 0.75))

print(paste("\nکوانتیل‌های تئوری: Q1=", quantiles_theo[1], 
           ", Q2=", quantiles_theo[2], ", Q3=", quantiles_theo[3]))
print(paste("کوانتیل‌های شبیه‌سازی: Q1=", quantiles_sim[1], 
           ", Q2=", quantiles_sim[2], ", Q3=", quantiles_sim[3]))

قضیه حد مرکزی (Central Limit Theorem)

قضیه حد مرکزی - CLT

قضیه حد مرکزی یکی از مهم‌ترین قضایای آمار است که بیان می‌کند میانگین نمونه‌ها از هر توزیعی (با میانگین و واریانس محدود) با افزایش اندازه نمونه به توزیع نرمال نزدیک می‌شود.

بیان ریاضی: اگر X₁, X₂, ..., Xₙ نمونه تصادفی از توزیع با μ و σ² باشند، آنگاه:

X̄ₙ ~ N(μ, σ²/n) برای n بزرگ (معمولاً n ≥ 30)

فرمول‌های کلیدی CLT:

میانگین نمونه: X̄ = (X₁ + X₂ + ... + Xₙ)/n

توزیع میانگین: E(X̄) = μ

واریانس میانگین: Var(X̄) = σ²/n

خطای معیار: SE = σ/√n

استاندارد: Z = (X̄ - μ)/(σ/√n) ~ N(0,1)

مثال: میانگین قد دانشجویان

قد دانشجویان با میانگین 170 سانتی‌متر و انحراف معیار 10 سانتی‌متر توزیع شده. نمونه‌ای 36 نفره انتخاب می‌کنیم.

سؤالات:

الف) توزیع میانگین نمونه چیست؟

ب) احتمال میانگین نمونه بین 168 تا 172؟

حل:

الف) X̄ ~ N(170, 10²/36) = N(170, 100/36) = N(170, 2.78)

ب) P(168 ≤ X̄ ≤ 172) = P(-1.2 ≤ Z ≤ 1.2) ≈ 0.770

تمرین توتوریوم A4: توزیع نرمال پیشرفته

پارامترها: X ~ N(μ = 5, σ² = 9)

الف) محاسبه P(X ≤ 2) و P(X ≤ 11)
ب) یافتن کوانتیل‌های 1%, 10%, 50%, 90%, 99%
ج) محاسبه P(0 ≤ X ≤ 2) و P(|X - μ| ≤ 3)
د) یافتن مقدار a برای P(X - μ < -a) = 0.05

حل کامل تمرین A4:

# تمرین توتوریوم A4: توزیع نرمال X ~ N(5, 9)
# Tutorial Exercise A4: Normal Distribution

# پارامترهای توزیع
mu <- 5
sigma <- 3  # انحراف معیار (ریشه واریانس)

# الف) محاسبه احتمالات
# P(X ≤ 2)
prob_x_leq_2 <- pnorm(2, mean = mu, sd = sigma)
print(paste("P(X ≤ 2) =", round(prob_x_leq_2, 4)))

# P(X ≤ 11)
prob_x_leq_11 <- pnorm(11, mean = mu, sd = sigma)
print(paste("P(X ≤ 11) =", round(prob_x_leq_11, 4)))

# ب) کوانتیل‌های مختلف
quantiles <- c(0.01, 0.10, 0.50, 0.90, 0.99)
quantile_values <- qnorm(quantiles, mean = mu, sd = sigma)

for(i in 1:length(quantiles)) {
    print(paste("x", quantiles[i], "=", round(quantile_values[i], 3)))
}

# ج) احتمالات پیچیده‌تر
# P(0 ≤ X ≤ 2)
prob_0_to_2 <- pnorm(2, mu, sigma) - pnorm(0, mu, sigma)
print(paste("P(0 ≤ X ≤ 2) =", round(prob_0_to_2, 4)))

# P(|X - μ| ≤ 3) = P(2 ≤ X ≤ 8)
prob_within_3 <- pnorm(8, mu, sigma) - pnorm(2, mu, sigma)
print(paste("P(|X - μ| ≤ 3) =", round(prob_within_3, 4)))

# د) یافتن مقدار a
# P(X - μ < -a) = 0.05
a_value <- abs(qnorm(0.05, mean = 0, sd = sigma))
print(paste("a =", round(a_value, 4)))

# بررسی: P(X - μ > a) = 0.05
a_check <- qnorm(0.95, mean = 0, sd = sigma)
print(paste("a (بررسی) =", round(a_check, 4)))

# تمرین توتوریوم A5: آزمون عملکرد
# Tutorial Exercise A5: Performance Test

# پارامترهای توزیع
mu_test <- 150
sigma_test <- 36

# الف) رسم تابع چگالی
x_vals <- seq(mu_test - 4*sigma_test, mu_test + 4*sigma_test, 1)
y_vals <- dnorm(x_vals, mean = mu_test, sd = sigma_test)

plot(x_vals, y_vals, type = "l", lwd = 2, col = "blue",
     main = "Dichtefunktion der Normalverteilung N(150, 36²)",
     xlab = "x", ylab = "f(x)",
     xlim = c(30, 270), ylim = c(0, max(y_vals)*1.1))

# نمایش میانگین
abline(v = mu_test, col = "red", lty = 2, lwd = 2)
text(mu_test, max(y_vals)*0.8, "μ = 150", pos = 4)

# ب1) P(X ≤ 140) و نمایش سطح
prob_140 <- pnorm(140, mean = mu_test, sd = sigma_test)
print(paste("P(X ≤ 140) =", round(prob_140, 4)))

# نمایش سطح زیر منحنی
x_fill_140 <- seq(30, 140, 1)
y_fill_140 <- dnorm(x_fill_140, mu_test, sigma_test)
polygon(c(30, x_fill_140, 140), c(0, y_fill_140, 0), 
         col = "lightblue", border = NA)

# ب4) P(130 < X < 200)
prob_130_200 <- pnorm(200, mu_test, sigma_test) - pnorm(130, mu_test, sigma_test)
print(paste("P(130 < X < 200) =", round(prob_130_200, 4)))

# ج) کوانتیل 10%
x_10_percent <- qnorm(0.1, mean = mu_test, sd = sigma_test)
print(paste("10%-Quantil =", round(x_10_percent, 1)))
print("10% der Testteilnehmer haben kleiner (gleich) 103.86 Punkte.")

تمرین‌های کاربردی و مسائل حل شده

مجموعه جامع تمرین‌های عملی

این بخش شامل مسائل کاربردی از انواع توزیع‌های مختلف است که با روش گام به گام حل شده‌اند.

مسئله 1: کنترل کیفیت تولید (توزیع نرمال)

کارخانه‌ای قطعاتی با میانگین وزن 100 گرم و انحراف معیار 5 گرم تولید می‌کند.

الف) احتمال قطعه‌ای بین 95 تا 105 گرم؟

ب) حد کنترل کیفیت برای رد 5% قطعات؟

حل مسئله 1:

X ~ N(100, 5²)

الف) P(95 ≤ X ≤ 105) = Φ(1) - Φ(-1) ≈ 0.683

ب) حد بالا: 100 + 5×1.645 ≈ 108.2 گرم

حد پایین: 100 - 5×1.645 ≈ 91.8 گرم

# حل کامل مسائل کاربردی
# Complete solution for applied problems

# مسئله 1: کنترل کیفیت
mu_quality <- 100
sigma_quality <- 5

# الف) احتمال بین 95 تا 105
prob_range <- pnorm(105, mu_quality, sigma_quality) - 
              pnorm(95, mu_quality, sigma_quality)
print(paste("P(95 ≤ X ≤ 105) =", round(prob_range, 3)))

# ب) حدود کنترل کیفیت
upper_limit <- qnorm(0.975, mu_quality, sigma_quality)
lower_limit <- qnorm(0.025, mu_quality, sigma_quality)
print(paste("حد بالا:", round(upper_limit, 1), "گرم"))
print(paste("حد پایین:", round(lower_limit, 1), "گرم"))
